1. 푸드 코드 

A마트 푸트 코드에는 상하이, 한성면옥, 전주한식, 돈이돈가스 음식코너가 있다.
음식은 주문전용기계를 통해 주문하고 쿠폰을 받아 교환하는 방식으로 받는다.

요구사항
나는 주말에 아이들과 A마트 푸드코드를 자주 이용한다.
A마트 푸드 코트에는 상하이, 한성면억, 전주한식, 돈이돈가스 음식코드가 있고,
각각 다섯개 이상의 메뉴가 있다. 나와 아이들은 각자 원하는 음식을 주문기계로 주문한다.
큰아이는 두가지 음식을 시키기도 한다.
주문을 한 후 주문번호가 적은 쿠폰을 가지고 기다리면 음식이 나온다. 음식은 쿠폰과 교환하여 받아온다.

2. 학사관리

대학은 학사관리 시스템을 구축하여 학사 정보를 체계적으로 관리한다.

요구사항
ㅇ 학상은 담당교수와 소속학과가 있어야 하고, 수강신천을 하여 강좌를 수강해야 한다. 그리고 모든 학새은 학생번호를 부여하여 식별하며 학생이름, 학생주민번호, 학생주소, 학생전화번호, 학생이메일, 학생사진, 소속학과번호 정보를 가진다.
0 교수는 소속학과가 있어야 하고 학생을 담당할 수도 있어야 하며 강좌에 대한 강의를 해야 한다.
그리고 모든 교수는 교수번호를 부여하여 식별하며 교수이름, 교수주민번호, 교수주소, 교수전화번호, 교수이메일, 교수사진 소속학과번호 정보를 가진다.
교수와 학생은 학생번호와 교수번호를 이용한 관계를 갖고 현재 학년/학기 정보를 포함한다.
0 학과는 교수와 학생들로 이루어졌으며 해당학과에 대한 강좌가 있어야 한다.
그리고 학과 학과번호를 부여하여 식별하며 학과명 학과전화번호 정보를 갖는다.
0 강좌는 강의하는 교수가 있어야 하지만 간혹 강의가 진행되지 않은 강좌들도 있을 수 있다.
그리고 학생이 수강을 해야 하고 강좌를 진행할 학과가 있어야 한다. 또한 강좌는 강좌번호와 담당교수번호를 부여하여 식별하며 강좌명, 취득학점, 강의시간, 강의실정보를 갖는다.
0 수강(수강내역)은 학생번호, 강좌번호, 교수번호를 부여하여 식별하며 출석점수, 중간고사점수, 기말고사점수, 기타점수, 총점, 평점 정보를 갖는다.

3. 은행 업무관리

은행은 지점별로 기본적인 은행업무를 수행한다.
고객은 통장을 가지고 입출금이나 카드 서비스등을 이용하고 직원은 고객 업무를 도와 은행 내부 업무를 수행한다.
은행은 고객과 내부 직원의 원할한 업무를 위해 데이터베이스를 구축하고자 한다.

요구사항

0 예금계좌는 고객의 통장을 관리하기 위한 개체다
0 예금계좌 거래내역은 각 예금계좌에 대한 입출급 내역을 관리하기 위한 개체다.
0 하나의 입금이나 출금 내역은 반드시 거래건별로 날짜와 함께 계좌에 남긴다.
0 은행은 예금및 카드 서비스를 고객에게 제공한다.
0 고객은 고객주민번호를 부여하여 식별하며 고객명, 고객주소, 고객생년월일, 고객이메일, 고객전화번호, 고객직업등의 정보를 가진다.
0 카드(업무)는 고유의 카드ID를 부여하여 식별하며 카드신청일자, 카드한도금액, 카드결재일자, 카드종류, 고객주민번호, 예금계좌ID 정보를 가진다.
0 예금계좌는 예금계좌ID를 부여하여 식별하며 예금계좌종류, 예금잔고, 카드신청여부, 예금개설일자, 예금자이름, 전화번호, 이메일, 고객주민번호 정보를 포함한다.
0 예금계좌 거래내역은 예금계좌ID, 입출금날짜, 거래번호를 부여하여 식별하며 예금구분, 예금내용, 거래금액, 예금잔고 정보를 가진다.

4. 캠핑카 대여관리

캠핑카 회사에서는 캠핑카 중계사이트를 열어 여러 캠핑카 대여 회사가 캠핑카를 등록하고 고객들이 대여할 수 있도록 할 계획이다.
캠핑카의 특수성을 고려하여 캠핑카 정비업체와도 연계할 계획이다.
이사이트에서 구축할 데이터베이스는 캠핑카 대여회사, 캠핑카 정비업소, 고객을 연결해 주는 역할을 할 것이다.

요구사항
0 캠핑카 대여회사는 캠핑카대여회사ID를 부여하고 회사명, 주소, 전화번호, 담당자이름, 담당자이메일 정보를 등록한다.
0 캠핑카는 캠핑카등록ID를 부여받고, 캠핑카이름, 캠핑타차량번호, 캠핑카승차인원수, 캠핑카이미지, 캠핑카상세정보, 캠핑카대여비용 정보를 포함한다. 또한 어느 캠핑카 대여 회사의 소유인지 알기 위해 캠핑카대여회사Id와 캠핑카등록일자를 등록한다.
0 고객은 고객이 보유한 운전면허증번호를 등록하고, 고객명, 고객주소, 고객전화번호,고객이메일 이전에 캠핑카 사용날짜 및 캠핑카 종류 정보를 등록한다.
0 고객은 원하는 캠핑카를 대여할 수 있다.
캠핑카 대여을 하게 되면 고유 대여번호를 부여받고 캠핑카등록id, 운전면허증번호, 캠핑카대여회사ID, 대여시작일, 청구요금, 납입기한, 기타청구내역, 기타청구요금 정보를 등록한다.
0 캠핑카 정비소는 캠핑카정비소id 부여받고, 정비소명, 정보소주소, 정비소전화번호, 담당자 이름 및 이메일 정보를 등록한다.
0 고객이 특정 캠핑카 정비소에 정비를 의뢰하면 정비정보를 남긴다.
정비정보는 고유 정비번호를 부여받고, 캠핑카등록ID, 캠핑카대여회사ID,고객운전면허증번호, 정비내역, 수리날짜, 수리비용, 납입기한, 기타정비 내역정보를 등록한다.
